# URL Shortener Frontend

Authenticated URL shortener frontend built with React, Vite, Redux Toolkit, and TailwindCSS.

## Features

- **Authentication**: Login and Signup with form validation.
- **Dashboard**: View, create, and delete short URLs.
- **Copy to Clipboard**: Easily copy short URLs.
- **Analytics**: View click counts for each URL.
- **Responsive Design**: Works on desktop and mobile.
- **Theme**: Dark mode interface.

## Tech Stack

- React
- Vite
- TypeScript
- Redux Toolkit (State Management)
- React Router DOM
- TailwindCSS (Styling)
- Shadcn/UI (Design System implementation)
- React Hook Form + Zod (Validation)
- Lucide React (Icons)

## Prerequisites

- Node.js (v18+)

## Setup

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Run Development Server**

   ```bash
   npm run dev
   ```

   The app will run at `http://localhost:5173`.

   > **Note**: Ensure the backend is running at `http://localhost:9000` for API requests to work.

## Build

```bash
npm run build
```
